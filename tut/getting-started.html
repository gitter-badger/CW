Getting Started

Dependencies

CW will currently only run on MAC OS X or Linux (i.e. POSIX) systems. This means it is currently unavailable for users of Windows Systems. This may change in the future.

CW is currently tested on Ruby versions v1.9.3, v2.0.0 and v2.2.3

Install Ruby

A fairly recent version of the Ruby programming language (1.8.7 or later) is required to run CW. See Ruby Installation for details of downloading the latest version. To check Ruby has installed type the following to check the version:


ruby --version

Install CW RubyGem

Enter the following commands to create the directory cw (in the home directory) - enter the directory - and install the CW gem and its dependencies.


mkdir ~/cw
cd ~/cw
gem install cw

How to Run the Examples:

Download (or create) a test-script in the directory created. Run a test-script by typing ruby followed by the name of the text file or test-script that you wish to run:

download: five-common-words.rb


ruby five-common-words.rb
